# 연산자
> ### **단항연산자**
  - **+/-**
  - **증감연산자**
    <pre>a=3;
    a++; //a값은 4가 된다.</pre>
  - **casting연산자(형변환)**
    <pre>float a=3;
    int b=(int)a;
    //앞에서 a를 float형으로 선언했으니 int형인 b에 a를 대입하려면 a를 int형으로 형변환 해주어야 한다.</pre>  
> ### **이항연산자**
  - **산술연산자** <br /> : +, -, *, /, %
  - **관계연산자(비교)** <br />: >, >=, <, <=, ==, !=
  - **논리연산자**(각각 순서대로 AND, OR, XOR, NOT)
    - 논리(피연산자 논리형) : &&, ||, ^, !
    - 비트(정수) : &, |, ^, ~
> ### **삼항연산자**
  <pre>식 ? 참 : 거짓;</pre>
